## React & TypeScript Features

I used React functional components with hooks like useState and useEffect to manage state and side effects. TypeScript was used to define strong types for tasks, props, and state, which helped catch errors earyly and keep components consistent.

## Challenges & How I Solved Them 

One challenge was handling edit vs create logic in the form. I solved it by passing intialData as a prop and syncing it with local state using useEffect.
